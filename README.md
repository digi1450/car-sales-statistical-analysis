# Car Sales Statistical Analysis

## Project Overview
This project performs statistical analysis on a car sales dataset using Python.  
The analysis includes data cleaning, exploratory data analysis (EDA), descriptive statistics, and inferential statistics such as confidence intervals and hypothesis testing.

## Dataset
- File: `Carsale.csv`
- Records: 23,906 rows
- Variables: Customer information, car specifications, price, income, transmission type, and dealer region.

## Project Structure
├── Carsale.csv          # Dataset
├── analysis.py          # Main analysis script
├── requirements.txt     # Python dependencies
└── README.md            # Project documentation
